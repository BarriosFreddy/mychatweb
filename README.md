# Welcome to My Chat Web

My chat web is an app to communicate with your friends, relatives and more. 

[APP DEMO](https://epic-northcutt-592ff3.netlify.app/)

## Stack

 - MongoDB
 - Express
 - React JS
 - Node JS

## Used Libraries
For backend
 - Mongoose
 - Body-parser
 - Cors
 - Socket-io

For frontend
 - Axios
 - React Bootstrap
 - React Icons
 - Firebase Storage Service
 - History
 - React Router Dom
 - Socket-io client

 
## Tools

 - Nodemon
 - Yarn

## Development

### Backend
To run the backend of the project, It is mandatory to execute the following commands in the root folder.

    $ cd backend

To download dependencies

    $ yarn install
To deploy project

    $ yarn start
